# Focusboard

Focusboard is a small, personal task-management panel for planning the day, keeping projects moving, and protecting focus time. It runs entirely in the browser and saves the board on the current device.

## What it includes

- A daily overview with task counts, priorities, project progress, and weekly rhythm.
- A task list where tasks can be added, completed, moved between stages, or removed.
- A compact project board for planned work, work in progress, and review.
- A 25-minute focus timer tied to a selected task.
- Local browser storage, so there is no account or server to set up.

## Run locally

Open `index.html` in a modern web browser. No installation or build command is needed.

## Testing

Open the panel in a browser and check that task creation, completion, deletion, status changes, navigation, focus timing, and local saving work correctly.
